# Scheme — маленький интерпретатор на C++20

В этом учебном проекте я реализовал интерпретатор небольшого подмножества языка Scheme.  
Он умеет читать выражение из строки, разбирать его в AST, вычислять встроенные функции и печатать результат обратно в Scheme-формате.

---
## Что поддерживается

### Синтаксис
- Целые числа (внутри — int64_t)
- Символы.
- Булевые переменные
- Списки
- Пары (dotted pairs)
- Пустой список
- (quote X) — возвращает X “как есть”, не вычисляя

### Встроенные функции (реализованы в src/builtins)

#### Quote
- quote

#### Boolean
- boolean?
- not
- and (short-circuit)
- or  (short-circuit)

#### Integer
- number?
- сравнения: =, <, <=, >, >=  
  (работают на списке аргументов, например (< 1 2 3))
- арифметика: +, -, *, /
- max, min, abs

#### Pair/List
- предикаты: pair?, null?, list?
- базовые операции: cons, car, cdr
- создание списка: list
- доступ: list-ref, list-tail

## Как проект работает

**Tokenizerer** (src/tokenizer.cpp)  
   Разбивает входную строку на токены: числа, символы, ( ), точку, кавычку.

**Parserer** (src/parser.cpp)  
   Строит AST в виде объектов:
   - Number — число
   - Symbol — символ
   - Cell — пара (first, second) для списков/пар
   - nullptr используется как пустой список ()

**Scheme** (src/scheme.cpp)  
   Если выражение — число/символ, оно просто возвращается.
   Если выражение — Cell, считается что это вызов функции:
   - берём имя из first
   - ищем обработчик в таблице builtins (src/builtins/builtins.cpp)
   - вызываем нужную функцию
